Reidworks

Skills Library

Open, reusable skills for Claude. In Claude Code, one install loads them all; in Cowork, add the ones you want as ZIPs. Use them yourself or fork them for your own setup.

Install in Claude Code (recommended)

/plugin marketplace add reidworks-io/reidworks-skills
/plugin install reidworks-core@reidworks-skills

One install loads every skill. Don't want one? Disable it in the /plugin menu after installing.

Install in Cowork (personal account)

No bulk install here — add each skill you want individually.

1. Download the ZIP for the skill you want from the catalog below
2. Cowork → Customize → Skills
3. + → + Create skill → Upload a skill
4. Upload the ZIP

Cowork's "Add marketplace" is org/team-only. ZIPs are auto-rebuilt on every push to main. Refs: Use skills · Use plugins.
